# Java Development

Java fundamentals coursework, labs, and final project from a Java Development course. Covers core OOP concepts through to web-connected Java applications.

## Contents

- **Labs** — Weekly lab exercises covering OOP, collections, I/O, concurrency
- **Final Project** — Full-stack Java web application
- **Reference PDFs** — Java How to Program (Deitel), Effective Java, Cracking the Coding Interview

## Tech Stack

- **Language**: Java
- **Frontend**: HTML/CSS/JavaScript (JSP/Servlets)
- **Tools**: Maven, Shell scripts
